Hosting a Webpage on a Linux Machine (Ubuntu) Using Apache2 This project involves setting up a basic web server on a Linux machine running Ubuntu using the Apache2 web server. The goal is to host a functional webpage and its accessable to devices within the local network for privacy and security.
