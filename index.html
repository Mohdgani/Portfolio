College project

(Jan 2022)

Hosting a Webpage on a Linux Machine (Ubuntu) Using Apache2. This project involves setting up a basic web server on a Linux machine running Ubuntu using the Apache2 web server. The goal is to host a functional webpage accessible to devices within the local network for privacy and security. The project includes installing Apache2, configuring the web server, placing webpage files in the appropriate directory, adjusting permissions of the firewall, implementing local network restrictions via configuration files, and testing access. It highlights foundational skills in web hosting, server management, and network security.
